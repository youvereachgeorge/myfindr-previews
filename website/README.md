# myFindr Website Sandbox

This is your staging environment for myfindr.com.au design changes.

## Workflow
1. Make changes here in VSCode
2. Test locally by opening index.html in browser
3. When ready → push to GitHub
4. George deploys to live site

## Live site
myfindr.com.au is hosted on Framer.
This folder is for HTML/CSS prototyping before applying to Framer.
