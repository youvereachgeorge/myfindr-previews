# myFindr Email Templates

## Folders
- `welcome.html` — New client welcome email (sent from admin portal)
- `previews/` — Open these in browser to see how emails look

## How to edit
1. Edit the HTML file in VSCode
2. Open in browser to preview
3. When happy, push to GitHub
4. The send script picks up changes automatically

## Variables used
- `{{CLIENT_NAME}}` — First name
- `{{CLIENT_EMAIL}}` — Login email
- `{{CLIENT_PASSWORD}}` — Temp password
- `{{ICP_TYPE}}` — Interior Designer / Builder / Real Estate
- `{{LOOM_URL}}` — Walkthrough video URL
